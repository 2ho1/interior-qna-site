📞 섹션 4: 견적 문의 섹션 (Quote Request Section)
┌─────────────────────────────────────────────────────────────────────────────┐
│ 목적: 잠재 고객의 견적 문의 유도 및 리드 생성                                 │
│ 구성요소:                                                                     │
│ • 섹션 제목: "무료 견적 문의"                                               │
│ • 문의 폼 구성:                                                             │
│                                                                             │
│ 필수 입력 항목:                                                             │
│ 1. 고객 이름 (텍스트 입력)                                                  │
│ 2. 연락처 (휴대폰 번호, 유효성 검사 포함)                                   │
│ 3. 주소 (주소 검색 API 연동 또는 직접 입력)                                 │
│ 4. 주거지 형태 (드롭다운):                                                  │
│    - 아파트                                                                 │
│    - 주택/빌라                                                              │
│    - 상가/사무실                                                            │
│    - 기타                                                                   │
│ 5. 평수 (숫자 입력, 단위: 평)                                               │
│ 6. 예상 예산 (드롭다운):                                                    │
│    - 500만원 미만                                                          │
│    - 500만원 ~ 1000만원                                                     │
│    - 1000만원 ~ 2000만원                                                    │
│    - 2000만원 ~ 3000만원                                                    │
│    - 3000만원 이상                                                          │
│ 7. 희망 시공일 (달력 선택기)                                                │
│ 8. 상세 요청사항 (텍스트 영역, 선택사항)                                    │
│                                                                             │
│ • 제출 버튼: "무료 견적 문의하기"                                           │
│ • 개인정보 처리방침 동의 체크박스                                           │
│ • 이메일 발송 확인: EmailJS 연동                                            │
└─────────────────────────────────────────────────────────────────────────────┘

================================================================================
                            🛠️ 기술 스택 및 개발 환경
================================================================================

💻 프론트엔드
• HTML5, CSS3, JavaScript (ES6+)
• CSS Framework: Bootstrap 5 또는 Tailwind CSS
• 아이콘: Font Awesome 또는 Heroicons
• 폰트: Google Fonts (한글 지원)

📧 백엔드 서비스
• EmailJS (이메일 발송 서비스)
• 클라이언트 사이드에서 직접 이메일 발송
• 별도 서버 구축 불필요

🌐 배포 환경
• Vercel (무료 플랜 사용)
• GitHub 연동을 통한 자동 배포
• 커스텀 도메인 설정 가능

🎨 UI/UX 라이브러리
• 캐러셀: Swiper.js 또는 Bootstrap Carousel
• 모달: Bootstrap Modal 또는 Custom Modal
• 날짜 선택: Flatpickr.js 또는 HTML5 Date Picker
• 폼 유효성 검사: JavaScript 또는 HTML5 Validation

📱 반응형 디자인
• 모바일 퍼스트 접근법
• 브레이크포인트:
  - 모바일: 320px ~ 767px
  - 태블릿: 768px ~ 1023px
  - 데스크톱: 1024px 이상
